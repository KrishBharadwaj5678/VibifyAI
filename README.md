# 🎧 Vibify AI 

An AI-powered emotion detection web app that analyzes facial expressions through your webcam and dynamically adapts music based on your mood.

![VibifyAIDemo](https://github.com/KrishBharadwaj5678/VibifyAI/raw/main/VibifyAIDemo.png)

## 🌟 Features

| Feature                        | Description                                |
| ------------------------------ | ------------------------------------------ |
| 😊 Real-Time Emotion Detection | Detects facial emotions instantly using AI |
| 🎵 Mood-Based Music            | Automatically changes music based on mood  |
| 📊 Emotion Analytics           | Live emotion tracking with charts          |
| 😄 Emoji Overlay               | Displays live emotion emoji on screen      |
| 🎯 Face Bounding Box           | AI face tracking with confidence score     |
| 📈 Emotion Timeline            | Visual mood changes over time              |
| 📱 Responsive Design           | Works across desktop and mobile devices    |
| ⚡ Fast Detection               | Real-time processing using Face API.js     |
