# 🎧 Vibify ИИ

Веб-приложение на базе ИИ для распознавания эмоций, которое анализирует выражение лица через веб-камеру и динамически подбирает музыку в зависимости от вашего настроения.

![VibifyAIDemo](https://github.com/KrishBharadwaj5678/VibifyAI/raw/main/VibifyAIDemo.png)

## 🌟 Возможности

| Функция                                    | Описание                                                  |
| ------------------------------------------ | --------------------------------------------------------- |
| 😊 Распознавание эмоций в реальном времени | Мгновенно определяет эмоции лица с помощью ИИ             |
| 🎵 Музыка по настроению                    | Автоматически меняет музыку в зависимости от настроения   |
| 📊 Аналитика эмоций                        | Отслеживание эмоций в реальном времени с графиками        |
| 😄 Emoji Overlay                           | Отображает эмодзи текущей эмоции на экране                |
| 🎯 Обнаружение лица                        | AI-отслеживание лица с показателем уверенности            |
| 📈 Таймлайн эмоций                         | Визуализация изменений настроения со временем             |
| 📱 Адаптивный дизайн                       | Работает на компьютерах и мобильных устройствах           |
| ⚡ Быстрое распознавание                    | Обработка в реальном времени с использованием Face API.js |

## 🛠️ Технологии

| Технология                | Использование                       |
| ------------------------- | ----------------------------------- |
| 🌐 HTML5                  | Структура и разметка                |
| 🎨 CSS tailwind           | Стилизация и адаптивный интерфейс   |
| ⚡ JavaScript (ES Modules) | Логика приложения                   |
| 🤖 Face API.js            | Распознавание эмоций по лицу        |
| 📊 Chart.js               | Визуализация аналитики эмоций       |
| 🎥 ВебRTC                 | Доступ к веб-камере                 |
| 🎵 API HTML-аудио         | Динамическое воспроизведение музыки |
