# 🎧 Vibify AI

ウェブカメラを通じて顔の表情を分析し、気分に応じて音楽を動的に変化させる、AI搭載の感情検出ウェブアプリです。

![VibifyAIDemo](https://github.com/KrishBharadwaj5678/VibifyAI/raw/main/VibifyAIDemo.png)

## 🌟 機能

| 機能            | 説明                     |
| ------------- | ---------------------- |
| 😊 リアルタイム感情検出 | AIを使用して顔の感情を瞬時に検出      |
| 🎵 気分に応じた音楽再生 | 感情に応じて自動で音楽を変更         |
| 📊 感情分析       | チャートによるリアルタイム感情追跡      |
| 😄 絵文字オーバーレイ  | 画面上に現在の感情の絵文字を表示       |
| 🎯 顔認識ボックス    | 信頼度スコア付きAI顔追跡          |
| 📈 感情タイムライン   | 時間経過による気分変化を可視化        |
| 📱 レスポンシブデザイン | デスクトップ・モバイル両対応         |
| ⚡ 高速検出        | Face API.jsによるリアルタイム処理 |

| 技術                        | 用途              |

## 🚀 インストール

### 1️⃣ リポジトリをクローン

```bash
git clone https://github.com/KrishBharadwaj5678/VibifyAI.git
```

### 2️⃣ プロジェクトフォルダへ移動

```bash
cd VibifyAI
```

### 3️⃣ ローカルで実行

`index.html` をブラウザで開くか、ローカルサーバー上で実行してください。
