# 🎧 Vibificar IA

Um aplicativo web com inteligência artificial para detecção de emoções que analisa expressões faciais através da sua webcam e adapta dinamicamente a música de acordo com o seu humor.

![Demonstração de IA do Vibify](https://github.com/KrishBharadwaj5678/VibifyAI/raw/main/VibifyAIDemo.png)

## 🌟 Funcionalidades

| Funcionalidade                       | Descrição                                             |
| ------------------------------------ | ----------------------------------------------------- |
| 😊 Detecção de Emoções em Tempo Real | Detecta emoções faciais instantaneamente usando IA    |
| 🎵 Música Baseada no Humor           | Altera automaticamente a música de acordo com o humor |
| 📊 Análise de Emoções                | Monitoramento de emoções em tempo real com gráficos   |
| 😄 Sobreposição de Emoji             | Exibe emojis de emoções ao vivo na tela               |
| 🎯 Caixa de Detecção Facial          | Rastreamento facial com IA e nível de confiança       |
| 📈 Linha do Tempo das Emoções        | Visualiza mudanças de humor ao longo do tempo         |
| 📱 Design Responsivo                 | Funciona em desktops e dispositivos móveis            |
| ⚡ Detecção Rápida                    | Processamento em tempo real usando Face API.js        |

## 🛠️ Tecnologias Utilizadas

| Tecnologia                | Uso                                 |
| ------------------------- | ----------------------------------- |
| 🌐 HTML5                  | Estrutura e layout                  |
| 🎨 Tailwind CSS           | Estilização e interface responsiva  |
| ⚡JavaScript (Módulos ES) | Lógica da aplicação                 |
| 🤖 Face API.js            | Detecção de emoções faciais         |
| 📊 Chart.js               | Visualização de análises emocionais |
| 🎥 WebRTC                 | Acesso à webcam                     |
| 🎵 API de áudio HTML       | Reprodução dinâmica de música       |
